# System Permissions

The Phoenix system is made up of three primary components, each with their own set of permissions and parameters:

* Fee Source - manages protocol-related fee logic
* Asset Sink - defines the eligible releaser
* Firepit (Releaser) - defines the economic criteria for releasing assets
